# Cliperton Website

## React + TypeScript + Vite

This is the official website for Cliperton, a modern clipboard manager application.

## Google Analytics Integration

This website includes comprehensive Google Analytics 4 tracking with the measurement ID `G-DSTCBX210L`. The following events are automatically tracked:

### Automatic Tracking (Enhanced Measurement)
- **Page views**: All page visits and navigation
- **Scroll tracking**: Automatic scroll depth measurement
- **File downloads**: Automatic download tracking
- **Outbound clicks**: Links to external websites

### Custom Event Tracking
- **Download button clicks**: Track when users click download buttons
- **Navigation clicks**: Track menu navigation usage
- **Scroll milestones**: Custom tracking at 25%, 50%, 75%, and 100% scroll depths
- **Button interactions**: Track CTA button clicks in Hero section

### Implementation Details
- **Analytics utilities**: `src/utils/analytics.ts` - Central tracking functions
- **Scroll tracking hook**: `src/hooks/useScrollTracking.ts` - Automatic scroll depth tracking
- **Component integration**: Download, Hero, and Header components include event tracking
